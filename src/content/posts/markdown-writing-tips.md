## 为什么选择 Markdown

Markdown 是一种轻量级标记语言，让你专注于内容而不是格式。它简洁、易学、跨平台。

## 标题层级

合理使用标题层级可以让文章结构更清晰：

- `#` 用于文章主标题
- `##` 用于主要章节
- `###` 用于子章节

不建议跳级使用标题，比如从 `##` 直接到 `####`。

## 代码块

使用三个反引号包裹代码块，并指定语言：

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## 列表使用

有序列表适合表示步骤：

1. 打开终端
2. 进入项目目录
3. 运行命令

无序列表适合并列内容：

- 简洁
- 易读
- 跨平台

## 引用

使用 `>` 来添加引用：

> 写作是思考的过程，Markdown 让这个过程更加顺畅。

## 链接和图片

合理使用链接可以丰富文章内容，但不要过度使用，影响阅读流畅性。

## 总结

掌握这些基本技巧，你就能写出清晰易读的 Markdown 文档了。

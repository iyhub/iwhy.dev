## 静态站点生成器

### [Astro](https://astro.build)
本博客使用的框架，零 JS 默认输出，内容优先的设计理念非常适合博客场景。

### [Next.js](https://nextjs.org)
React 生态最流行的框架，App Router 带来了全新的开发体验。

### [Hugo](https://gohugo.io)
Go 语言编写的静态站点生成器，构建速度极快，适合大型站点。

## 设计资源

### [Tailwind CSS](https://tailwindcss.com)
原子化 CSS 框架，v4 版本带来了更好的性能和开发体验。

### [霞鹜文楷](https://github.com/lxgw/LxgwWenKai)
开源的中文字体，优雅的楷体风格，非常适合阅读场景。

## 部署平台

### [Vercel](https://vercel.com)
前端部署的首选平台，与 Next.js 深度集成。

### [Cloudflare Pages](https://pages.cloudflare.com)
免费额度慷慨，全球 CDN 加速，性能优秀。

## 本周思考

搭建个人博客是一个很好的学习项目，既能实践技术，又能养成写作习惯。选择适合自己的工具，然后开始写作吧。
